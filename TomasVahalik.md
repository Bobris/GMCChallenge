OpenSource Lover Quest
======================

Favourite repositories
----------------------
### [MSOpenTech/opencv](https://github.com/MSOpenTech/opencv)
Fork of official computer vision library with ability to build for Windows Phone 8 or Win RT.

### [david-sabata/web-scrobbler](https://github.com/david-sabata/web-scrobbler)
Extension for scrobbling music played in web browser to last.fm. Every music lover should have it!

### [vrana/adminer](https://github.com/vrana/adminer)
Best lightweight database management tool for php.

### [Raynos/leaked-handles](https://github.com/Raynos/leaked-handles)
Awesome for tests in node.js - reveals handles that keep process open. :) Saves time.

### [gulpjs/gulp](https://github.com/gulpjs/gulp)
Streaming build system in node.js - awesome for automatization of daily tasks.

### [nwjs/nw.js](https://github.com/nwjs/nw.js)
Building desktop applications with web technologies.
